🔧 Configuração de Wi-Fi

Como restringir por Wi-Fi da empresa:

No arquivo index.html, localize:
const COMPANY_WIFI_IPS = [];

Substitua pelo prefixo do IP do roteador. Exemplo:
const COMPANY_WIFI_IPS = ['192.168.1'];

Como descobrir o IP: No celular conectado ao Wi-Fi da empresa, acesse https://api.ipify.org e copie os 3 primeiros grupos do número.

Para a câmera funcionar o sistema precisa estar em HTTPS. Use o GitHub Pages (grátis) para hospedar.

Passo a passo:
1. Crie conta em github.com
2. Crie repositório público
3. Suba os arquivos index.html e admin.html
4. Settings → Pages → branch main → Save
5. Acesse via https://seuusuario.github.io/repositorio
